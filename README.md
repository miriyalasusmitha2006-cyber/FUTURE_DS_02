# FUTURE_DS_02 – Customer Retention & Churn Analysis

## 📊 Project Overview

This project focuses on Customer Retention & Churn Analysis using the Telco Customer Churn dataset.

The objective is to identify:

* customer churn patterns
* retention trends
* customer lifetime behavior
* business insights to reduce churn

---

## 🛠 Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* VS Code

---

## 📁 Dataset

Dataset Used:
WA_Fn-UseC_-Telco-Customer-Churn.csv

Source:
Telco Customer Churn Dataset (Kaggle)

---

## 📈 Analysis Performed

* Data Cleaning
* Churn Rate Calculation
* Retention Rate Analysis
* Contract-based Churn Analysis
* Tenure Analysis
* Internet Service Impact Analysis
* Monthly Charges vs Churn

---

## 📊 Key Insights

* Month-to-month contract customers have the highest churn rate.
* Customers with lower tenure are more likely to leave.
* Higher monthly charges are associated with increased churn.
* Fiber optic users show relatively higher churn behavior.

---

## 💡 Recommendations

* Improve onboarding experience for new customers.
* Encourage long-term subscription contracts.
* Create loyalty and retention programs.
* Optimize pricing strategies for high-risk customers.

---

## 📷 Output Visualizations

The project generates multiple business visualizations:

* Churn by Contract Type
* Churn by Tenure Group
* Churn by Internet Service
* Monthly Charges vs Churn

---

## 🚀 Outcome

This analysis helps businesses understand customer behavior and implement strategies to improve retention and reduce churn.
